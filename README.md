# Surrogate_Internal_Standard_Analysis
Enhanced Quantification of 64 Pesticides in Cannabis-Derived Oil by Isotope-Dilution LC-MS/MS at Regulatory Decision Thresholds

Use the Iterative Curve Building file to make curves

Use the MAB Calculations file to apply curves to QC data

Each .py has its own README, and an example running command is listed at the top of the .py files.
